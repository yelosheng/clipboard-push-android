# 剪贴板推送 App - 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
Clipboard Push (暂定)

### 1.2 产品定位
一款简洁的 Android 剪贴板推送工具，实现手机与 Flask 后端之间的内容双向传输。

### 1.3 核心价值
- 从服务器接收推送内容，自动写入手机剪贴板
- 从手机分享内容到服务器

### 1.4 参考竞品
- Pushbullet

---

## 2. 目标用户
需要在手机与服务器/电脑之间快速传递文本、图片、文件的用户。

---

## 3. 功能需求

### 3.1 接收推送（服务器 → App）

| 功能项 | 说明 |
|--------|------|
| 支持内容类型 | 文本、图片、文件 |
| 连接方式 | 保持持久连接，确保实时接收（技术方案待定） |
| 接收后行为 | 自动写入系统剪贴板，用户可立即粘贴 |
| 通知 | 收到推送时显示通知提醒 |

### 3.2 发送内容（App → 服务器）

| 功能项 | 说明 |
|--------|------|
| 触发方式 | 通过系统分享菜单，选择本 App 进行分享 |
| 支持内容类型 | 文本、图片、文件 |
| 发送目标 | Flask 后端服务器 |

### 3.3 通知栏常驻

| 功能项 | 说明 |
|--------|------|
| 常驻通知 | 显示连接状态，保持后台服务运行 |

## 5. 系统架构

```
┌─────────────────┐                  ┌─────────────────┐
│                 │   Push (实时)    │                 │
│   Flask 后端    │ ───────────────► │   Android App   │
│                 │                  │                 │
│                 │ ◄─────────────── │                 │
│                 │   Share (HTTP)   │                 │
└─────────────────┘                  └─────────────────┘
                                            │
                                            ▼
                                     ┌─────────────┐
                                     │ 系统剪贴板  │
                                     └─────────────┘
```

**数据流向：**
1. **服务器 → App**：后端推送内容 → App 实时接收 → 写入系统剪贴板
2. **App → 服务器**：用户分享内容 → 选择本 App → 上传至后端

## 6. 技术实现

### 6.1 服务器信息

| 项目 | 详情 |
|------|------|
| 默认端口 | 9661 |
| WebSocket 端点 | `ws://<服务器地址>:9661/ws` (原生 WebSocket) |

### 6.2 API 接口

#### 发送文本
```
POST /api/push/text
Content-Type: application/json

{"content": "文本内容"}
```

#### 发送文件
```
POST /api/push/file
Content-Type: multipart/form-data

file: [文件二进制]
```

#### 下载文件
```
GET /files/<filename>
```

### 6.3 WebSocket 消息格式

接收到的消息为 JSON 格式：

```json
{
  "id": "uuid-string",
  "type": "text | image | video | audio | file",
  "content": "消息内容或文件描述",
  "file_url": "/files/xxx.jpg",
  "file_name": "original_name.jpg",
  "file_size": 12345,
  "mime_type": "image/jpeg",
  "timestamp": "2026-01-29T16:00:00"
}
```

### 6.4 Android 端技术方案

| 模块 | 方案 |
|------|------|
| WebSocket 客户端 | OkHttp WebSocket |
| 后台保活 | 前台服务 (Foreground Service) + 常驻通知 |
| 网络请求 | OkHttp / Retrofit |
| 文件下载 | OkHttp + 保存到缓存目录 |
| 剪贴板操作 | ClipboardManager |
| 系统分享接收 | Intent Filter (ACTION_SEND) |
